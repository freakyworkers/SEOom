# AWS EC2 인스턴스 유형 선택 가이드

**상황:** Free Tier 대상인지 아닌지에 따라 선택이 달라져요

---

## 📋 인스턴스 유형 선택 가이드

### Free Tier 대상인 경우

**조건:**
- AWS 계정을 처음 만든 경우
- 계정 생성 후 12개월 이내
- 이전에 EC2를 사용한 적이 없는 경우

**선택:**
- ✅ **t2.micro** 또는 **t3.micro** 선택
- ✅ **"Free tier 사용 가능"** 표시가 보여요
- 💰 **비용:** 무료 (월 750시간까지)

### Free Tier 대상이 아닌 경우

**조건:**
- 이전에 AWS를 사용한 적이 있는 경우
- 계정 생성 후 12개월이 지난 경우
- 이미 Free Tier를 사용한 적이 있는 경우

**선택:**
- ✅ **t3.micro** 선택 (가장 작고 저렴한 유형)
- ⚠️ **"Free tier 사용 가능"** 표시가 안 보여요
- 💰 **비용:** 약 월 7,000원~10,000원 (사용량에 따라 다름)

---

## 💰 비용 정보

### t3.micro 비용 (Free Tier 대상이 아닌 경우)

**서울 리전 기준:**
- **인스턴스 비용:** 약 시간당 $0.0104 (약 14원)
- **월 예상 비용:** 약 7,000원~10,000원 (24시간 실행 시)
- **데이터 전송:** 처음 1GB 무료, 이후 GB당 약 1,000원

**비용 절약 팁:**
1. **사용하지 않을 때는 인스턴스 중지**
   - 중지하면 인스턴스 비용은 발생하지 않아요
   - 스토리지 비용만 발생해요 (약 월 2,000원)
2. **필요할 때만 시작**
   - 개발/테스트 중일 때만 실행
   - 완료되면 중지

## ✅ 선택 체크리스트

### Free Tier 확인 방법

1. **"Free tier 사용 가능"** 표시가 보이나요?
   - ✅ 보임 → Free Tier 대상! t2.micro 또는 t3.micro 선택
   - ❌ 안 보임 → Free Tier 대상 아님! t3.micro 선택

### 인스턴스 유형 비교

| 유형 | vCPU | RAM | Free Tier | 월 예상 비용 |
|------|------|-----|-----------|------------|
| t2.micro | 1 | 1GB | ✅ 가능 | 무료 (Free Tier) |
| t3.micro | 2 | 1GB | ✅ 가능 | 무료 (Free Tier) 또는 약 7,000원 |

## 🎯 추천 선택

### 상황 1: "Free tier 사용 가능" 표시가 보이는 경우
- ✅ **t2.micro** 또는 **t3.micro** 선택
- 둘 다 무료예요!
- t3.micro가 조금 더 빠르지만, t2.micro도 충분해요

### 상황 2: "Free tier 사용 가능" 표시가 안 보이는 경우
- ✅ **t3.micro** 선택
- 가장 작고 저렴한 유형이에요
- 사용하지 않을 때는 반드시 중지하세요!

---

## ⚠️ 주의사항

### 비용 관리

1. **인스턴스 중지 방법:**
   - EC2 콘솔 → 인스턴스 선택 → "인스턴스 상태" → "중지"
   - 중지하면 인스턴스 비용은 발생하지 않아요

2. **인스턴스 삭제 방법:**
   - 완전히 삭제하려면 "종료" 선택
   - ⚠️ 삭제하면 데이터가 사라져요!

3. **비용 알림 설정:**
   - AWS 콘솔 → "청구" → "예산" 설정
   - 월 예산을 설정하면 초과 시 알림을 받을 수 있어요

## 🆘 질문

**Q: t3.micro가 너무 비싸요. 더 저렴한 방법이 있나요?**
A: 
- 사용하지 않을 때는 반드시 중지하세요 (스토리지 비용만 발생)
- Lightsail을 사용하는 방법도 있어요 (월 약 5,000원)
- 하지만 EC2가 더 유연해요

**Q: Free Tier를 다시 받을 수 있나요?**
A: 
- 아니요, Free Tier는 계정당 한 번만 받을 수 있어요
- 새 계정을 만들면 다시 받을 수 있지만, 카드 정보가 필요해요

**Q: 인스턴스를 중지하면 데이터가 사라지나요?**
A: 
- 아니요, 중지하면 데이터는 그대로 있어요
- 다시 시작하면 이전 상태 그대로 돌아와요
- 삭제(종료)하면 데이터가 사라져요
