### 🔴 1단계: Laravel 프로젝트 초기화 (필수)

현재 코드 파일만 있고 Laravel 프로젝트가 초기화되지 않았습니다.

#### 1-1. Laravel 프로젝트 생성 또는 초기화

**옵션 A: 기존 디렉토리에 Laravel 설치**
```bash
# 현재 디렉토리에서
composer create-project laravel/laravel . --prefer-dist
# 기존 파일들과 충돌 주의!
```

**옵션 B: 새 디렉토리에 Laravel 생성 후 파일 이동**
```bash
# 새 디렉토리에 Laravel 생성
composer create-project laravel/laravel SEOom_new

# 기존 파일들을 새 프로젝트로 복사
# (app/, database/, resources/, routes/, config/ 등)
```

#### 1-2. 필수 파일 확인/생성

- ✅ `composer.json` - Laravel 의존성
- ✅ `artisan` - Laravel CLI
- ✅ `.env.example` - 환경 변수 템플릿
- ✅ `bootstrap/` - 부트스트랩 파일
- ✅ `public/` - 공개 디렉토리
- ✅ `storage/` - 저장소 디렉토리
- ✅ `vendor/` - Composer 패키지

#### 1-3. 의존성 설치
```bash
composer install
npm install  # (필요시)
```

#### 1-4. 환경 설정
```bash
# .env 파일 생성
cp .env.example .env

# 애플리케이션 키 생성
php artisan key:generate

# .env 파일 수정
# - DB_CONNECTION=mysql
# - DB_DATABASE=seoom
# - DB_USERNAME=root
# - DB_PASSWORD=
```

#### 1-5. 데이터베이스 마이그레이션
```bash
php artisan migrate
php artisan db:seed --class=MasterUserSeeder
```
