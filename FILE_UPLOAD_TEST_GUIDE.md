## 테스트 방법

### 1. 게시글 작성 페이지 접속
- URL: `http://localhost:8000/site/test-site/boards/free/posts/create`
- 로그인 상태여야 합니다.

### 2. 게시글 작성
- 제목: "파일 업로드 테스트 게시글"
- 내용: "이 게시글은 PNG, GIF, WEBP 파일 업로드 테스트를 위한 것입니다."

### 3. 파일 업로드
- "첨부파일" 필드에서 위의 3개 파일을 모두 선택
- 여러 파일을 동시에 선택할 수 있습니다 (Ctrl 키를 누른 채 클릭)

### 4. 게시글 작성
- "작성" 버튼 클릭

### 5. 결과 확인
- 게시글 상세 페이지에서 첨부파일이 정상적으로 표시되는지 확인
- 각 첨부파일을 클릭하여 다운로드가 정상적으로 작동하는지 확인
- 이미지 파일의 경우 미리보기가 표시되는지 확인

## 지원되는 파일 형식
- **이미지**: PNG, JPG, JPEG, GIF, WEBP
- **문서**: PDF, DOC, DOCX, XLS, XLSX
- **압축**: ZIP

## 파일 크기 제한
- 최대 10MB까지 업로드 가능

## 확인 사항
- [ ] PNG 파일 업로드 성공
- [ ] WEBP 파일 업로드 성공
- [ ] GIF 파일 업로드 성공
- [ ] 여러 파일 동시 업로드 성공
- [ ] 첨부파일 다운로드 정상 작동
- [ ] 이미지 파일 미리보기 정상 작동

## 문제 해결
- 파일 업로드가 실패하는 경우:
  1. 파일 크기가 10MB를 초과하지 않는지 확인
  2. 파일 형식이 지원되는 형식인지 확인
  3. 브라우저 콘솔에서 오류 메시지 확인
  4. Laravel 로그 파일 확인 (`storage/logs/laravel.log`)
