## 📋 1단계: 서버 업데이트

서버에 접속된 상태에서 다음 명령어를 **하나씩** 입력하세요:

```bash
sudo apt update
```

Enter 키 누르고, 완료되면:

```bash
sudo apt upgrade -y
```

**예상 시간:** 2-5분

---

## 💡 팁

- 명령어는 **하나씩** 입력하세요
- 각 명령어가 완료될 때까지 기다리세요
- `sudo` 명령어는 비밀번호를 물을 수 있어요 (없으면 그냥 Enter)

---

## 🆘 문제 해결

### 문제: 명령어가 실행되지 않아요

**해결 방법:**
- 서버에 접속된 상태인지 확인 (`ubuntu@ip-172-31-38-145:~$` 프롬프트가 보여야 해요)
- 명령어를 정확히 입력했는지 확인

### 문제: 비밀번호를 물어봐요

**해결 방법:**
- AWS EC2는 기본적으로 비밀번호가 없어요
- 그냥 Enter 키를 누르면 돼요
