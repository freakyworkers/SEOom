# PowerShell 붙여넣기 경고 해결

**상황:** 여러 줄 명령어를 붙여넣을 때 경고 메시지가 나와요

---

## ✅ 해결 방법: "붙여넣기" 버튼 클릭!

### 경고 메시지 설명

**경고 내용:**
- "여러 줄이 있는 텍스트를 붙여 넣습니다"
- "이 텍스트를 셸에 붙여넣으면 예기치 않은 명령 실행이 발생할 수 있습니다"

**이건 뭔가요?**
- Windows Terminal/PowerShell의 보안 기능이에요
- 여러 줄 명령어를 붙여넣을 때 나오는 경고예요
- 우리가 제공한 명령어는 **완전히 안전**해요!

---

## ✅ 안전한 명령어 확인

우리가 제공한 명령어:
```powershell
icacls "C:\Users\kangd\Desktop\세움배포파일\seoom-key.pem" /inheritance:r
icacls "C:\Users\kangd\Desktop\세움배포파일\seoom-key.pem" /grant:r "%USERNAME%:R"
```

**이 명령어는:**
- ✅ 키 파일 권한만 설정하는 명령어예요
- ✅ 서버에 접속하기 위한 준비 작업이에요
- ✅ 완전히 안전한 명령어예요!

---

## 🎯 해결 방법

### 방법 1: "붙여넣기" 버튼 클릭 (권장)

1. 경고 창에서 **"붙여넣기"** 버튼 클릭
2. 명령어가 실행돼요
3. 완료!

### 방법 2: 한 줄씩 입력 (더 안전하게 느껴지면)

경고가 걱정되면 한 줄씩 입력할 수도 있어요:

**1번째 명령어:**
```powershell
icacls "C:\Users\kangd\Desktop\세움배포파일\seoom-key.pem" /inheritance:r
```
- 복사 → 붙여넣기 → Enter

**2번째 명령어:**
```powershell
icacls "C:\Users\kangd\Desktop\세움배포파일\seoom-key.pem" /grant:r "%USERNAME%:R"
```
- 복사 → 붙여넣기 → Enter

---

## 💡 왜 이 경고가 나오나요?

**이유:**
- Windows Terminal은 보안을 위해 여러 줄 명령어 붙여넣기를 확인해요
- 악의적인 명령어가 실행되는 것을 방지하기 위한 기능이에요
- 우리 명령어는 안전하니까 걱정하지 마세요!
