## 🎯 히어로 섹션

### 메인 헤드라인
```
10분 만에 나만의 커뮤니티를 만드세요
```

### 서브 헤드라인
```
코딩 없이, 복잡한 설정 없이
SEOom Builder로 전문적인 웹사이트를 시작하세요
```

### 주요 포인트 (3개)
- ✅ **10분 완성** - 복잡한 설정 없이 즉시 시작
- ✅ **무제한 커스터마이징** - 나만의 디자인과 기능
- ✅ **완전한 멀티테넌트** - 여러 사이트를 한 번에 관리

### CTA 버튼
- **주요 CTA**: "무료로 시작하기" (큰 버튼, 강조 색상)
- **보조 CTA**: "데모 보기" (아웃라인 버튼)

### 히어로 이미지/비디오
- 대시보드 스크린샷 또는 짧은 데모 비디오

## 💎 핵심 가치 제안

### 섹션 제목
```
왜 SEOom Builder를 선택해야 할까요?
```

### 3가지 핵심 가치

#### 1. 🚀 빠른 시작
**제목**: 10분 만에 사이트 완성
**설명**: 
- 복잡한 서버 설정이나 코딩 지식이 필요 없습니다
- 간단한 가입 후 바로 사이트를 만들 수 있습니다
- 자동 Provisioning으로 즉시 사용 가능합니다

**아이콘/이미지**: ⚡ 또는 시계 아이콘

#### 2. 🎨 완전한 커스터마이징
**제목**: 나만의 디자인과 기능
**설명**:
- 위젯 시스템으로 자유롭게 레이아웃 구성
- 커스텀 페이지로 원하는 콘텐츠 추가
- 다크 모드, 반응형 디자인 자동 지원

**아이콘/이미지**: 🎨 또는 브러시 아이콘

#### 3. 🏢 멀티테넌트 관리
**제목**: 여러 사이트를 한 번에
**설명**:
- 하나의 계정으로 여러 사이트 관리
- 마스터 콘솔에서 모든 사이트 통합 관리
- SSO 로그인으로 편리한 접근

**아이콘/이미지**: 🏢 또는 네트워크 아이콘

## ⚡ 주요 기능 소개

### 섹션 제목
```
강력한 기능으로 완벽한 커뮤니티를 만드세요
```

### 기능 카드 그리드 (2열 또는 3열)

#### 1. 게시판 시스템
**아이콘**: 📝
**제목**: 다양한 게시판 타입
**설명**:
- 일반 게시판, 클래식 게시판, 사진 게시판, 북마크 게시판
- 주제(Topic) 시스템으로 체계적인 분류
- Summernote 에디터로 풍부한 콘텐츠 작성

#### 2. 댓글 시스템
**아이콘**: 💬
**제목**: 대댓글 지원 커뮤니케이션
**설명**:
- 계층형 댓글 구조
- 실시간 알림 시스템
- 추천/비추천 기능

#### 3. 회원 관리
**아이콘**: 👥
**제목**: 강력한 회원 시스템
**설명**:
- 소셜 로그인 지원 (구글, 카카오 등)
- 포인트 시스템
- 회원 등급 관리
- 출석 체크 기능

#### 4. 위젯 시스템
**아이콘**: 🧩
**제목**: 자유로운 레이아웃 구성
**설명**:
- 메인 위젯, 사이드바 위젯, 커스텀 페이지 위젯
- 드래그 앤 드롭으로 쉬운 배치
- 다양한 위젯 타입 제공

#### 5. 관리자 도구
**아이콘**: ⚙️
**제목**: 강력한 관리 기능
**설명**:
- 직관적인 관리자 대시보드
- 게시판/게시글/회원 통합 관리
- 사이트 설정 완전 제어
- 통계 및 분석 기능

#### 6. SEO 최적화
**아이콘**: 🔍
**제목**: 검색 엔진 최적화
**설명**:
- 자동 sitemap.xml 생성
- robots.txt 관리
- RSS 피드 지원
- 메타 태그 커스터마이징

#### 7. 보안 기능
**아이콘**: 🔒
**제목**: 안전한 데이터 보호
**설명**:
- 멀티테넌트 데이터 완전 격리
- IP 차단 기능
- 신고 및 제재 시스템
- SSL/TLS 암호화

#### 8. 커스텀 도메인
**아이콘**: 🌐
**제목**: 나만의 도메인 연결
**설명**:
- 서브도메인 자동 제공
- 커스텀 도메인 연결 지원
- Cloudflare 연동으로 빠른 속도

### 섹션 제목

### FAQ 항목

#### Q1. 코딩 지식이 없어도 사용할 수 있나요?
**A**: 네, 완전히 가능합니다. SEOom Builder는 드래그 앤 드롭 방식의 직관적인 인터페이스를 제공하여 누구나 쉽게 사이트를 만들 수 있습니다.

#### Q2. 무료 플랜에도 모든 기능이 포함되나요?
**A**: 무료 플랜에는 기본 기능이 포함되어 있으며, 더 많은 기능이 필요하시면 유료 플랜으로 업그레이드할 수 있습니다.

#### Q3. 커스텀 도메인을 연결할 수 있나요?
**A**: 프로 플랜 이상에서는 커스텀 도메인을 연결할 수 있습니다. 간단한 DNS 설정만으로 연결이 완료됩니다.

#### Q4. 데이터 백업은 어떻게 되나요?
**A**: 모든 플랜에서 자동 백업이 제공되며, 엔터프라이즈 플랜에서는 더 자주 백업됩니다.

#### Q5. 여러 사이트를 관리할 수 있나요?
**A**: 네, 하나의 계정으로 여러 사이트를 만들고 관리할 수 있습니다. 마스터 콘솔에서 모든 사이트를 한눈에 볼 수 있습니다.

#### Q6. 환불 정책은 어떻게 되나요?
**A**: 30일 무조건 환불 보장 정책을 제공합니다. 만족하지 않으시면 전액 환불해드립니다.
