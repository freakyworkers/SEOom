# 가비아 네임서버 설정 가이드 (Cloudflare 연동)

**도메인:** 가비아에서 구매한 도메인  
**목적:** Cloudflare와 연결하기

---

## ⚠️ 중요: 구매 시점과 변경 시점이 달라요!

### 도메인 구매할 때:
- ✅ **"가비아 네임서버 사용"** 선택
- 아직 Cloudflare 설정이 안 되어 있어서요!

### Cloudflare 설정 완료 후:
- ✅ **"타사 네임서버 사용"** 선택
- Cloudflare 네임서버 주소를 입력해야 해요!

---

## ✅ 정답: Cloudflare 설정 후 "타사 네임서버 사용" 선택!

Cloudflare를 사용하려면 **반드시 "타사 네임서버 사용"**으로 변경해야 해요!

---

## 📋 단계별 가이드

### 1단계: Cloudflare에서 네임서버 주소 확인

1. Cloudflare 대시보드 접속: https://dash.cloudflare.com/
2. 도메인 선택
3. 오른쪽에 **"네임서버"** 섹션 확인
4. 두 개의 네임서버 주소 복사
   - 예: `alice.ns.cloudflare.com`
   - 예: `bob.ns.cloudflare.com`

### 2단계: 가비아에서 네임서버 설정

1. **가비아 홈페이지 접속**
   - https://www.gabia.com/

2. **로그인**

3. **"마이 가비아" 메뉴 클릭**
   - 상단 메뉴에서 찾기

4. **"도메인" 메뉴 클릭**
   - 왼쪽 메뉴에서 찾기

5. **구매한 도메인 찾기**
   - 도메인 목록에서 찾기

6. **"관리" 또는 "설정" 버튼 클릭**

7. **"네임서버 설정" 또는 "DNS 설정" 메뉴 찾기**
   - 보통 "도메인 설정" 또는 "고급 설정" 안에 있어요

8. **⚠️ "타사 네임서버 사용" 선택**
   - ❌ "가비아 네임서버 사용" 선택하지 마세요!
   - ✅ **"타사 네임서버 사용"** 선택!

9. **네임서버 입력란에 Cloudflare 네임서버 입력**
   - 첫 번째 네임서버: `alice.ns.cloudflare.com` (또는 Cloudflare에서 제공한 첫 번째)
   - 두 번째 네임서버: `bob.ns.cloudflare.com` (또는 Cloudflare에서 제공한 두 번째)

10. **"저장" 또는 "적용" 버튼 클릭**

11. **확인 메시지 확인**
    - "네임서버가 변경되었습니다" 같은 메시지가 나와요

### 3단계: 변경 확인

1. **24-48시간 대기** (보통 몇 시간이면 돼요)
2. Cloudflare 대시보드에서 확인
   - 도메인 옆에 **"활성"** 표시가 나오면 성공!
3. 또는 이메일로 확인
   - Cloudflare에서 "도메인이 활성화되었습니다" 이메일이 올 수 있어요

## ❓ 왜 "타사 네임서버 사용"을 선택해야 하나요?

- **"가비아 네임서버 사용"**: 가비아의 DNS 서버를 사용
  - Cloudflare의 기능을 사용할 수 없어요
  - SSL 인증서, 보안 기능 등을 사용할 수 없어요

- **"타사 네임서버 사용"**: Cloudflare의 네임서버를 사용
  - Cloudflare의 모든 기능을 사용할 수 있어요
  - 무료 SSL 인증서, 보안 기능, CDN 등을 사용할 수 있어요

---

## ⚠️ 주의사항

1. **네임서버 변경은 시간이 걸려요**
   - 보통 몇 시간에서 최대 48시간
   - 인내심을 가지세요!

2. **네임서버 주소를 정확히 입력하세요**
   - 오타가 있으면 작동하지 않아요
   - Cloudflare에서 제공한 주소를 정확히 복사하세요

3. **변경 중에는 도메인이 잠시 작동하지 않을 수 있어요**
   - 정상적인 현상이에요
   - 변경이 완료되면 다시 작동해요

---

## 🆘 문제 해결

### 문제: 네임서버 변경이 안 돼요

**해결 방법:**
1. 네임서버 주소가 정확한지 확인
2. 가비아 고객센터에 문의: 1588-4252

### 문제: 48시간이 지났는데도 안 돼요

**해결 방법:**
1. Cloudflare 대시보드에서 도메인 상태 확인
2. 네임서버 주소가 올바른지 다시 확인
3. 가비아에서 네임서버 설정이 저장되었는지 확인

---

## ✅ 완료 확인

네임서버 변경이 완료되면:

1. Cloudflare 대시보드에서 도메인 옆에 **"활성"** 표시 확인
2. 브라우저에서 도메인 주소 입력해보기
3. 사이트가 정상적으로 보이면 성공! 🎉
