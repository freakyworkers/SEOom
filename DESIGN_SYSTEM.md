# SEOom Builder 디자인 시스템

## 개요

SEOom Builder는 Bootstrap 5 기반의 일관된 디자인 시스템을 사용합니다. 이 디자인 시스템은 커뮤니티 사이트와 관리자 페이지, 그리고 향후 마스터 운영 콘솔에서도 재사용됩니다.

## 디자인 원칙

1. **일관성**: 모든 페이지에서 동일한 디자인 언어 사용
2. **접근성**: WCAG 2.1 AA 수준 준수
3. **반응형**: 모바일, 태블릿, 데스크톱 모든 기기 지원
4. **재사용성**: 컴포넌트 기반 구조로 유지보수 용이

## 레이아웃 구조

### 1. 커뮤니티 레이아웃 (`layouts/app.blade.php`)

- **네비게이션 바**: 상단 고정, Primary 색상
- **메인 콘텐츠**: Container 기반, 최대 너비 제한
- **푸터**: 하단 고정, 사이트 정보 표시

### 2. 관리자 레이아웃 (`layouts/admin.blade.php`)

- **사이드바**: 좌측 고정, 다크 테마
- **메인 콘텐츠**: 사이드바 옆 영역
- **반응형**: 모바일에서는 사이드바 숨김 처리

## 컴포넌트

### Alert 컴포넌트 (`components/alert.blade.php`)

```blade
<x-alert type="success">성공 메시지</x-alert>
<x-alert type="error">에러 메시지</x-alert>
<x-alert type="warning">경고 메시지</x-alert>
<x-alert type="info">정보 메시지</x-alert>
```

### Card 컴포넌트 (`components/card.blade.php`)

```blade
<x-card title="카드 제목">
    카드 내용
</x-card>
```

### Page Header 컴포넌트 (`components/page-header.blade.php`)

```blade
<x-page-header 
    title="페이지 제목"
    subtitle="부제목"
    :actions="'<button>액션 버튼</button>'"
/>
```

## 색상 시스템

- **Primary**: `#0d6efd` - 주요 액션, 링크
- **Success**: `#198754` - 성공 메시지, 긍정적 액션
- **Danger**: `#dc3545` - 삭제, 경고
- **Warning**: `#ffc107` - 주의, 고정 게시글
- **Info**: `#0dcaf0` - 정보성 메시지
- **Secondary**: `#6c757d` - 보조 요소

## 타이포그래피

- **제목**: H1-H6, Font-weight: 600
- **본문**: System font stack
- **작은 텍스트**: 0.875rem, text-muted

## 버튼 스타일

- **Primary**: 주요 액션
- **Secondary**: 보조 액션
- **Outline**: 덜 중요한 액션
- **Size**: sm, default, lg

## 테이블 스타일

- **헤더**: 대문자, 얇은 하단 보더
- **Hover**: 행 호버 효과
- **반응형**: 모바일에서는 스크롤 가능

## 카드 스타일

- **Shadow**: 기본 shadow-sm, 호버 시 shadow 증가
- **Border**: 없음 (shadow로 구분)
- **Hover**: 약간의 상승 효과

## 마스터 콘솔 디자인 재사용

마스터 운영 콘솔(`/master`)도 동일한 디자인 시스템을 사용합니다:

1. `layouts/master.blade.php` 생성 (admin.blade.php 기반)
2. 동일한 컴포넌트 사용
3. 색상 시스템 일관성 유지
4. 사이드바 구조 재사용
