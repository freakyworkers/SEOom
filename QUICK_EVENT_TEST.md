# 이벤트 게시판 빠른 테스트 가이드

## 🚀 테스트 시작하기

### 1. 접속 확인
- URL: http://localhost:8000
- 개발 서버가 실행 중입니다 ✅

### 2. 마이그레이션 실행 (필요시)
```bash
php artisan migrate
```

---

## 📝 빠른 테스트 시나리오

### 시나리오 1: 공지형 이벤트 (가장 간단)

1. **게시판 생성**
   - 관리자 → 게시판 관리 → 게시판 생성
   - 타입: **이벤트** 선택
   - 저장

2. **게시글 작성**
   - 게시판 → 글쓰기
   - 제목: "공지형 이벤트 테스트"
   - 이벤트 유형: **공지형 이벤트** 선택
   - 시작일/종료일 입력
   - 저장

3. **확인 사항**
   - ✅ 이벤트 정보 표시됨
   - ✅ **참여 버튼 없음** (공지 형태)

---

### 시나리오 2: 신청형 이벤트

1. **게시글 작성**
   - 이벤트 유형: **신청형 이벤트** 선택
   - 시작일/종료일 입력
   - 저장

2. **이벤트 신청**
   - "이벤트 신청하기" 버튼 클릭
   - AJAX로 처리되는지 확인

3. **확인 사항**
   - ✅ 버튼 클릭 시 "처리 중..." 표시
   - ✅ 신청 완료 후 메시지 표시
   - ✅ 참여자 수 증가
   - ✅ 중복 클릭 방지

4. **운영자 포인트 지급** (운영자 계정으로)
   - 게시글 하단 "운영자 기능" 섹션 확인
   - 포인트 입력 후 지급

---

### 시나리오 3: 정답형 이벤트 (가장 중요)

1. **게시글 작성**
   - 이벤트 유형: **정답형 이벤트** 선택
   - 퀴즈 선택지 추가:
     - 선택지 1: "정답입니다" ✅ (정답 체크)
     - 선택지 2: "오답입니다"
     - 선택지 3: "이것도 오답"
   - 저장

2. **정답 선택 테스트**
   - 라디오 버튼 클릭
   - ✅ **선택한 항목에 파란색 테두리/배경 표시**
   - ✅ **버튼 하단에 "선택한 정답: [텍스트]" 표시**

3. **정답 제출**
   - "정답 제출하기" 클릭
   - ✅ 정답/오답 결과 표시
   - ✅ 선택한 정답 텍스트 표시

4. **이미 참여한 경우**
   - 같은 게시글 다시 열기
   - ✅ "이미 이벤트에 참여하셨습니다" 표시
   - ✅ 선택한 정답 텍스트 표시

## 🐛 문제 발생 시 확인 사항

1. **마이그레이션 미실행**
   - `php artisan migrate` 실행

2. **JavaScript 오류**
   - 브라우저 콘솔 확인 (F12)
   - jQuery 로드 확인

3. **AJAX 오류**
   - 네트워크 탭에서 요청 확인
   - 서버 응답 확인

4. **권한 오류**
   - 로그인 상태 확인
   - 관리자 권한 확인
