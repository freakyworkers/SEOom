# 도메인 구매 가이드 (가비아)

**시점:** 도메인 구매할 때  
**중요:** 이 가이드는 **도메인 구매 시점**에 사용하는 거예요!

---

## 📋 도메인 구매 단계

### 1단계: 가비아 접속 및 로그인

1. https://www.gabia.com/ 접속
2. 로그인 (계정이 없으면 회원가입)

### 2단계: 도메인 검색

1. 상단 검색창에 원하는 도메인 입력 (예: `myseoom.com`)
2. **"검색"** 클릭
3. 사용 가능한 도메인 확인

### 3단계: 도메인 선택 및 장바구니 담기

1. 원하는 도메인 선택
2. 기간 선택 (보통 1년)
3. **"장바구니 담기"** 클릭

### 4단계: 결제 정보 입력

1. 장바구니에서 **"주문하기"** 클릭
2. 결제 정보 입력
3. **"결제하기"** 클릭

### 5단계: 네임서버 설정 (중요!)

⚠️ **여기서 중요한 선택을 해야 해요!**

**구매할 때는:**
- ✅ **"가비아 네임서버 사용"** 선택
- ❌ "타사 네임서버 사용" 선택하지 마세요!

**이유:**
- 아직 Cloudflare 설정이 안 되어 있어요
- Cloudflare 네임서버 주소를 모르니까요
- 나중에 Cloudflare 설정이 완료되면 변경할 거예요!

**화면 예시:**
```
┌─────────────────────────────────────┐
│ 네임서버 설정                        │
├─────────────────────────────────────┤
│ ● 가비아 네임서버 사용  ← 이거 선택!│
│ ○ 타사 네임서버 사용                │
└─────────────────────────────────────┘
```

### 6단계: 구매 완료

1. 결제 완료
2. 구매 완료 메시지 확인
3. 도메인 목록에서 확인

---

## ✅ 구매 완료 후

도메인 구매는 완료되었어요!

**다음 단계:**
1. AWS EC2 서버 만들기
2. 서버에 프로그램 설치하기
3. Cloudflare 설정하기
4. **그 다음에** 네임서버를 "타사 네임서버 사용"으로 변경하기

---

## 📝 요약

| 시점 | 선택 | 이유 |
|------|------|------|
| **도메인 구매할 때** | 가비아 네임서버 사용 | 아직 Cloudflare 설정 안 됨 |
| **Cloudflare 설정 후** | 타사 네임서버 사용 | Cloudflare 네임서버 주소 입력 |

---

## 🆘 질문

**Q: 구매할 때 "타사 네임서버 사용"을 선택하면 안 되나요?**  
A: 네, 안 돼요! Cloudflare 네임서버 주소를 모르는 상태에서는 입력할 수 없어요. 일단 "가비아 네임서버 사용"으로 구매하고, 나중에 변경하는 게 맞아요.

**Q: 나중에 변경하는 게 복잡하지 않나요?**  
A: 아니에요! 매우 간단해요. 가비아에서 "타사 네임서버 사용"으로 변경하고, Cloudflare 네임서버 주소만 입력하면 돼요. 2-3분이면 끝나요!
