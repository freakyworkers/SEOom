# ✅ 배포 체크리스트 (초보자용)

**단계별로 체크하면서 진행하세요!**

---

## 📋 1단계: 계정 만들기

- [ ] AWS 계정 만들기
  - [ ] 이메일로 가입
  - [ ] 카드 정보 입력 (1년 무료)
  - [ ] 전화번호 인증
  - [ ] 로그인 확인

- [ ] Cloudflare 계정 만들기
  - [ ] 이메일로 가입
  - [ ] 이메일 인증
  - [ ] 로그인 확인

- [ ] 도메인 구매 (선택)
  - [ ] 도메인 선택
  - [ ] 결제 완료

---

## 📋 2단계: AWS EC2 서버 만들기

- [ ] EC2 서비스 접속
- [ ] 리전 선택 (서울)
- [ ] 인스턴스 시작
  - [ ] 이름 입력
  - [ ] Ubuntu 22.04 선택
  - [ ] t2.micro 또는 t3.micro 선택 (무료)
  - [ ] 키 페어 생성 및 다운로드
  - [ ] 보안 그룹 설정 (SSH, HTTP, HTTPS)
  - [ ] 인스턴스 시작
- [ ] 퍼블릭 IP 주소 확인 및 복사

---

## 📋 3단계: 서버에 접속하기

- [ ] PuTTY 또는 WinSCP 다운로드
- [ ] 키 파일 변환 (.pem → .ppk)
- [ ] 서버 접속 성공 확인

---

## 📋 4단계: 프로그램 설치하기

- [ ] 시스템 업데이트
- [ ] PHP 8.3 설치
- [ ] Composer 설치
- [ ] MySQL 설치 및 데이터베이스 생성
- [ ] Apache 설치
- [ ] Git 설치

---

## 📋 5단계: 프로젝트 업로드하기

- [ ] 프로젝트 파일 업로드 (Git 또는 WinSCP)
- [ ] .env 파일 설정
- [ ] 의존성 설치
- [ ] 마이그레이션 실행
- [ ] 파일 권한 설정
- [ ] Apache 가상 호스트 설정

---

## 📋 6단계: Cloudflare 설정하기

- [ ] 도메인 추가
- [ ] 네임서버 변경 (도메인 구매 사이트에서)
- [ ] DNS 레코드 설정
  - [ ] A 레코드 (@)
  - [ ] A 레코드 (www)
  - [ ] 와일드카드 A 레코드 (*)
- [ ] SSL/TLS 설정 (전체 엄격 모드)
- [ ] 보안 설정

---

## 📋 7단계: 확인하기

- [ ] 도메인으로 접속 확인
- [ ] 마스터 로그인 확인
- [ ] 사이트 생성 테스트
- [ ] 서브도메인 접속 확인

---

## 📋 8단계: 보안 설정

- [ ] 마스터 계정 비밀번호 변경
- [ ] 방화벽 설정 확인
- [ ] 정기 업데이트 설정

---

## 🎉 완료!

모든 체크리스트를 완료하면 배포가 끝났어요!
