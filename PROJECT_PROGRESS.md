## 📁 주요 파일 구조

### 컨트롤러
```
app/Http/Controllers/
├── AdminController.php          # 관리자 대시보드, 사용자/게시판/게시글 관리
├── BoardController.php          # 게시판 관리 (생성/수정/삭제)
├── PostController.php           # 게시글 관리
├── CommentController.php        # 댓글 관리
├── SiteSettingController.php    # 사이트 설정
├── AuthController.php           # 인증
├── HomeController.php           # 홈
├── SearchController.php         # 검색
└── Master/                      # 마스터 콘솔
    ├── MasterAuthController.php
    ├── MasterDashboardController.php
    ├── MasterSiteController.php
    ├── MasterMonitoringController.php
    └── MasterBackupController.php
```

```
app/Models/
├── User.php                     # 사용자
```

```
│   ├── boards/

│   └── settings.blade.php
├── master/                      # 마스터 콘솔
```

## 📝 개발 규칙

### Git 커밋 메시지
```
feat: 기능 추가
fix: 버그 수정
docs: 문서 업데이트
refactor: 코드 리팩토링
style: 코드 스타일 변경
test: 테스트 추가
chore: 기타 작업
```

- PSR-12 코딩 표준 준수
- Service 중심 구조
