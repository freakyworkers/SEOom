# 🚀 토스 페이먼츠 API 키 빠른 가이드

## 현재 위치: API 키 페이지 ✅

이제 시크릿 키를 확인하고 복사하면 됩니다!

---

## 📋 단계별 안내

### 1️⃣ "API 개별 연동 키" 섹션 찾기
- 페이지를 아래로 스크롤하세요
- **"API 개별 연동 키"** 섹션을 찾습니다
- ⚠️ **"결제위젯 연동 키"가 아닙니다!** "API 개별 연동 키"를 사용해야 해요!

### 2️⃣ "테스트" 탭 확인
- 상단에 "테스트"와 "라이브" 탭이 있어요
- **"테스트"** 탭이 파란색으로 선택되어 있는지 확인하세요

### 3️⃣ 시크릿 키 확인하기
- **"시크릿 키"** 필드를 찾습니다
- 현재는 마스킹되어 있어요: `test_sk_************************`
- 시크릿 키 옆에 있는 **"보기"** 버튼을 클릭하세요!

### 4️⃣ 시크릿 키 복사하기
- "보기" 버튼을 클릭하면 전체 시크릿 키가 보입니다
- 시크릿 키 옆에 있는 **"복사"** 버튼을 클릭하세요
- 또는 키를 드래그해서 선택한 후 Ctrl+C로 복사하세요

---

## ✅ 다음 단계

시크릿 키를 복사했다면:

1. 우리 홈페이지로 돌아가기: `http://localhost:8000/master/login`
2. 마스터 콘솔 로그인
3. 왼쪽 메뉴에서 **"구독 설정"** 클릭
4. **"토스 페이먼츠 Secret Key"** 입력란에 붙여넣기
5. **"저장"** 버튼 클릭

---

- 지금은 **테스트 시크릿 키**를 사용하세요!
