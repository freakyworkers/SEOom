# 📝 SEOom Builder 메인 페이지 구성안 및 섹션별 문구

**프로젝트**: SEOom Builder (세움빌더)  
**대상**: seoomweb.com 마스터 사이트 메인 페이지  
**목적**: GPT를 통한 문구 보완 및 개선을 위한 기본 구성안  
**작성일**: 2025년 1월

---

## 📋 문서 사용 가이드

이 문서는 GPT에게 "다음 구성안을 바탕으로 각 섹션별 문구를 더 매력적이고 설득력 있게 보완해달라"는 프롬프트로 사용할 수 있습니다.

**GPT 프롬프트 예시**:
```
아래 SEOom Builder 메인 페이지 구성안을 참고하여, 각 섹션별 문구를 더 매력적이고 설득력 있게 보완해주세요. 
각 섹션의 목적과 톤앤매너를 유지하면서도, 사용자의 행동을 유도할 수 있는 문구로 개선해주세요.
```

## 🎯 전체 페이지 구성 개요

### 섹션 순서 및 목적

```
1. 히어로 섹션 (Hero Section)
   → 첫인상 형성, 핵심 가치 제안, 즉각적인 행동 유도

2. 핵심 가치 제안 (Value Proposition)
   → 경쟁 우위 강조, 사용자 이점 명확화

3. 주요 기능 소개 (Features)
   → 상세 기능 소개, 제품 강점 시각화, 신뢰 구축

4. 통계/사회적 증명 (Social Proof)
   → 신뢰도 향상, 사회적 증명 제공, 플랫폼 성장 강조

5. 요금제 안내 (Pricing)
   → 요금제 정보 제공, 가격 투명성, 플랜 선택 유도

6. 사이트 생성 CTA (Call to Action)
   → 최종 행동 유도, 회원가입/사이트 생성 전환

7. FAQ 및 문의 (FAQ & Contact)
   → 일반 질문 답변, 고객 지원 접근성, 신뢰도 구축
```

### 전체 레이아웃 구조

```
┌─────────────────────────────────────────┐
│  헤더 (사이트 설정)                      │
├─────────────────────────────────────────┤
│  [섹션 1] 히어로 섹션 (1열, 전체 높이)   │
│  └─ Block 위젯                          │
├─────────────────────────────────────────┤
│  [섹션 2] 핵심 가치 제안 (3열)          │
│  └─ Block 위젯 3개                      │
├─────────────────────────────────────────┤
│  [섹션 3] 주요 기능 소개 (2열)          │
│  ├─ Block 위젯 (기능 설명)              │
│  └─ Image Slide 위젯 (스크린샷)         │
├─────────────────────────────────────────┤
│  [섹션 4] 통계/사회적 증명 (4열)        │
│  └─ Block 위젯 4개                     │
├─────────────────────────────────────────┤
│  [섹션 5] 요금제 안내 (1열)             │
│  └─ Plans 위젯                         │
├─────────────────────────────────────────┤
│  [섹션 6] 사이트 생성 CTA (1열)         │
│  └─ Create Site 위젯                   │
├─────────────────────────────────────────┤
│  [섹션 7] FAQ 및 문의 (2열)            │
│  ├─ Custom HTML 위젯 (FAQ)              │
│  └─ Contact Form 위젯                  │
├─────────────────────────────────────────┤
│  푸터 (사이트 설정)                      │
└─────────────────────────────────────────┘
```

## 📝 섹션 1: 히어로 섹션 (Hero Section)

### 섹션 목적
- 방문자의 첫인상 형성
- 핵심 가치 제안 전달
- 즉각적인 행동 유도 (CTA)

### 톤앤매너
- **친근하고 접근하기 쉬운** 톤
- **구체적이고 명확한** 메시지
- **긍정적이고 에너지 넘치는** 표현

### 현재 문구 (기본안)

#### 메인 헤드라인
```
코딩 없이 지금 필요한 홈페이지를 만드세요
```

#### 서브 헤드라인
```
코딩 없이, 복잡한 설정 없이
SEOom Builder로 전문적인 랜딩페이지,브랜드/회사 홈페이지 , 커뮤니티까지!
```

#### 주요 포인트 (3개)
```
✅ 코딩 없이 위젯으로 - 복잡한 설정 없이 즉시 시작
✅ 무제한 커스터마이징 - 나만의 디자인과 기능
✅ 완전한 멀티테넌트 - 여러 사이트를 한 번에 관리
✅ 무료 SSL인증 - SSL인증서를 무상으로 제공
```

#### CTA 버튼 문구
```
무료로 시작하기
```

### 보완이 필요한 부분
- [ ] 헤드라인을 더 감정적으로, 더 강렬하게
- [ ] 서브 헤드라인에 구체적인 이점 추가
- [ ] 주요 포인트를 더 구체적이고 설득력 있게
- [ ] CTA 버튼 문구를 더 긴급성 있게

### 참고사항
- SEOom Builder는 **코딩 없이 커뮤니티/홈페이지를 만들 수 있는** SaaS 플랫폼
- **멀티테넌트 구조**로 여러 사이트를 한 번에 관리 가능
- **드래그 앤 드롭 위젯 시스템**으로 쉬운 커스터마이징

## 📝 섹션 7: FAQ 및 문의 (FAQ & Contact)

### 섹션 목적
- 일반적인 질문에 대한 답변 제공

### 톤앤매너
- **친절하고 도움이 되는** 톤

- **긍정적이고 안심시키는** 표현

### FAQ 제목
```

```

### FAQ 항목 1
**질문**: 코딩 지식이 없어도 사용할 수 있나요?

**답변**: 네, 완전히 가능합니다. SEOom Builder는 드래그 앤 드롭 방식의 직관적인 인터페이스를 제공하여 누구나 쉽게 사이트를 만들 수 있습니다.

### FAQ 항목 2
**질문**: 무료 플랜에도 모든 기능이 포함되나요?

**답변**: 무료 플랜에는 기본 기능이 포함되어 있으며, 더 많은 기능이 필요하시면 유료 플랜으로 업그레이드할 수 있습니다.

### FAQ 항목 3
**질문**: 커스텀 도메인을 연결할 수 있나요?

**답변**: 프로 플랜 이상에서는 커스텀 도메인을 연결할 수 있습니다. 간단한 DNS 설정만으로 연결이 완료됩니다.

### FAQ 항목 4
**질문**: 데이터 백업은 어떻게 되나요?

**답변**: 모든 플랜에서 자동 백업이 제공되며, 엔터프라이즈 플랜에서는 더 자주 백업됩니다.

### FAQ 항목 5
**질문**: 여러 사이트를 관리할 수 있나요?

**답변**: 네, 하나의 계정으로 여러 사이트를 만들고 관리할 수 있습니다. 마스터 콘솔에서 모든 사이트를 한눈에 볼 수 있습니다.

### FAQ 항목 6
**질문**: 환불 정책은 어떻게 되나요?

**답변**: 30일 무조건 환불 보장 정책을 제공합니다. 만족하지 않으시면 전액 환불해드립니다.

## 🎨 전체적인 보완 방향

### 1. 문구 개선 포인트
- **구체성**: 추상적인 표현보다 구체적인 숫자와 사실
- **이점 중심**: 기능 설명보다 사용자가 얻을 이점 강조
- **감정적 어필**: 논리적 설명에 감정적 메시지 추가
- **행동 유도**: 명확한 액션 제시

### 2. 톤앤매너 통일
- 전체 페이지에서 **일관된 톤앤매너** 유지
- **친근하면서도 전문적인** 균형
- **긍정적이고 에너지 넘치는** 표현

### 3. SEO 고려사항
- 주요 키워드 자연스럽게 포함
