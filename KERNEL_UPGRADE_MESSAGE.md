# 커널 업그레이드 메시지 해결

**상황:** `sudo apt upgrade -y` 실행 후 커널 업그레이드 메시지가 나왔어요

---

## ✅ 해결 방법: Ok 버튼 클릭!

### 메시지 설명

**"Pending kernel upgrade"** 메시지는:
- ✅ 정상적인 메시지예요
- ✅ 새로운 커널 버전이 설치되었다는 의미예요
- ✅ 지금은 재부팅할 필요 없어요

### 해야 할 것

1. **"Ok" 버튼 클릭** (또는 Enter 키)
2. 계속 진행하세요!

---

## 💡 재부팅에 대해

### 지금 재부팅할 필요 없어요!

**이유:**
- 아직 프로그램 설치가 남아있어요
- 모든 설치가 완료된 후에 재부팅하면 돼요
- 지금은 그냥 Ok를 누르고 계속 진행하세요

### 나중에 재부팅하는 방법

모든 설치가 완료된 후:

```bash
sudo reboot
```

또는 AWS 콘솔에서 인스턴스를 재시작할 수도 있어요.
