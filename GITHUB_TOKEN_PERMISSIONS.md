# GitHub Personal Access Token 권한 선택 가이드

**상황:** GitHub Personal Access Token 생성 중 권한 선택

---

## ✅ 권장 설정: 최소 권한만 선택

### 필수 권한

**저장소 (Repository)** 체크:
- ✅ **저장소 (Repository)** - 이것만 체크하면 돼요!
- 이 권한으로 private 저장소 클론이 가능해요

---

## ❌ 불필요한 권한들

다음 권한들은 **체크하지 않아도 돼요**:
- 워크플로 (Workflow)
- 패키지 (Package)
- 조직 (Organization)
- 기업 (Enterprise)
- GitHub Action
- GitHub Copilot
- GPG 키
- SSH 서명 키
- 기타 모든 권한들

---

## 💡 왜 최소 권한만?

### 보안상 좋아요:
- 필요한 권한만 부여하는 것이 안전해요
- 토큰이 유출되어도 피해를 최소화할 수 있어요
- "최소 권한 원칙"을 따르는 것이 좋아요

### 충분해요:
- `repo` 권한만으로도 private 저장소 클론이 가능해요
- 추가 권한은 필요 없어요

---

## 📋 올바른 설정

### 체크해야 할 것:
- ✅ **저장소 (Repository)** - 이것만!

### 체크하지 않아도 되는 것:
- ❌ 나머지 모든 권한들
