# 🎯 토스 페이먼츠 API 키 받는 방법 (초등학생도 이해하기 쉬운 설명)

## 📚 먼저 알아야 할 것

**토스 페이먼츠**는 우리가 만든 홈페이지에서 결제를 받을 수 있게 해주는 회사예요.
마치 **계산대**처럼 생각하면 됩니다! 
- 손님이 물건을 사면 → 계산대에서 돈을 받아요
- 우리 홈페이지에서도 → 토스 페이먼츠가 돈을 받아줘요

## 🔑 API 키란?

**API 키**는 마치 **열쇠**와 같아요!
- 집에 들어가려면 → 열쇠가 필요해요
- 토스 페이먼츠를 사용하려면 → API 키가 필요해요

**Secret Key**는 **비밀 열쇠**예요. 아무에게도 보여주면 안 되는 중요한 열쇠입니다!

## 📝 단계별 가이드 (쉽게 따라하기)

### 1단계: 토스 페이먼츠 홈페이지 가기 🏠

1. 인터넷 브라우저를 엽니다 (크롬, 엣지 등)
2. 주소창에 입력: **https://www.tosspayments.com**
3. 엔터를 누릅니다

### 2단계: 회원가입하기 👤

1. 화면 오른쪽 위에 **"회원가입"** 또는 **"무료로 시작하기"** 버튼을 찾습니다
2. 버튼을 클릭합니다
3. 필요한 정보를 입력합니다:
   - 이메일 주소 (예: myemail@example.com)
   - 비밀번호 (잘 기억해두세요!)
   - 이름, 전화번호 등
4. **"가입하기"** 버튼을 클릭합니다

### 3단계: 이메일 인증하기 ✉️

1. 가입할 때 입력한 이메일 주소로 가세요
2. 토스 페이먼츠에서 보낸 이메일을 찾습니다
3. 이메일 안에 있는 **"인증하기"** 버튼을 클릭합니다
4. 인증이 완료되면 로그인할 수 있어요!

### 4단계: 로그인하기 🔐

1. 토스 페이먼츠 홈페이지로 돌아갑니다
2. 오른쪽 위 **"로그인"** 버튼을 클릭합니다
3. 이메일과 비밀번호를 입력합니다
4. **"로그인"** 버튼을 클릭합니다

### 5단계: 테스트 모드로 시작하기 🧪

처음에는 **"테스트 모드"**로 시작합니다. 
- 테스트 모드 = 연습 모드 (진짜 돈이 빠져나가지 않아요!)
- 실제 모드 = 진짜 모드 (진짜 돈이 빠져나가요)

**테스트 모드에서 연습하는 것을 추천합니다!**

### 6단계: API 키 찾기 🔍

1. 왼쪽 사이드바에서 **"API 키"** 메뉴를 클릭합니다
2. 페이지가 로드되면 **"API 개별 연동 키"** 섹션을 찾습니다
   - "결제위젯 연동 키"가 아니라 **"API 개별 연동 키"**를 사용해야 해요!
3. **"테스트"** 탭이 선택되어 있는지 확인합니다 (파란색으로 표시됨)

### 7단계: Secret Key 확인하기 👁️

1. **"API 개별 연동 키"** 섹션에서 **"시크릿 키"** 필드를 찾습니다
2. 시크릿 키는 보안을 위해 마스킹되어 있어요: `test_sk_************************`
3. 시크릿 키 옆에 있는 **"보기"** 버튼을 클릭합니다
4. 클릭하면 마스킹이 해제되고 전체 시크릿 키가 보입니다!

### 8단계: Secret Key 복사하기 📋

1. **"보기"** 버튼을 클릭해서 시크릿 키가 보이게 한 후
2. 시크릿 키 옆에 있는 **"복사"** 버튼을 클릭합니다
   - 또는 시크릿 키를 마우스로 드래그해서 선택한 후 Ctrl+C (복사)
3. 이제 Secret Key가 클립보드에 복사되었어요!

**⚠️ 중요: 이 키는 절대 다른 사람에게 보여주면 안 됩니다!**

## 🎨 우리 홈페이지에 입력하기

### 방법 1: 마스터 콘솔에서 입력 (추천!)

1. 우리 홈페이지의 **마스터 콘솔**에 로그인합니다
   - 주소: `http://localhost:8000/master/login`
2. 왼쪽 메뉴에서 **"구독 설정"**을 클릭합니다
3. **"토스 페이먼츠 Secret Key"** 입력란을 찾습니다
4. 아까 복사한 Secret Key를 붙여넣기 합니다 (Ctrl+V)
5. **"저장"** 버튼을 클릭합니다
6. 완료! 이제 결제 기능을 사용할 수 있어요! 🎉

### 방법 2: 직접 데이터베이스에 입력 (고급 사용자용)

이 방법은 개발자만 사용하세요!

## 🧪 테스트 결제 해보기

### 테스트 카드 번호

테스트 모드에서는 진짜 카드가 아니라 **테스트 카드 번호**를 사용합니다:

**성공하는 테스트 카드:**
- 카드번호: `1234-5678-9012-3456`
- 유효기간: `12/34` (미래 날짜 아무거나)
- CVC: `123` (3자리 숫자 아무거나)
- 비밀번호: `123456` (6자리 숫자)

**실패하는 테스트 카드:**
- 카드번호: `1111-1111-1111-1111` (이 카드로 결제하면 실패해요)

### 테스트 방법

1. 우리 홈페이지 메인 페이지에 가세요
2. **"요금제"** 위젯을 찾습니다
3. 원하는 플랜의 **"구독하기"** 버튼을 클릭합니다
4. 결제 페이지에서 위의 테스트 카드 번호를 입력합니다
5. **"결제하기"** 버튼을 클릭합니다
6. 성공하면 "결제 완료" 메시지가 나와요!

## ❓ 자주 묻는 질문 (FAQ)

### Q1: API 키가 안 보여요
**A:** 로그인을 제대로 했는지 확인하세요. 그리고 "개발자센터" 또는 "내 상점" 메뉴를 찾아보세요.

### Q2: Secret Key를 잃어버렸어요
**A:** 토스 페이먼츠에 로그인해서 다시 확인할 수 있어요. 또는 "API 키 재발급" 기능을 사용하세요.

### Q3: 테스트 모드와 실제 모드의 차이는?
**A:** 
- **테스트 모드**: 연습용, 진짜 돈이 빠져나가지 않음
- **실제 모드**: 진짜 결제, 진짜 돈이 빠져나감

### Q4: 실제 모드로 바꾸려면?
**A:** 토스 페이먼츠에서 **"상점 인증"**을 완료해야 해요. 사업자 등록증 등을 제출해야 합니다.
