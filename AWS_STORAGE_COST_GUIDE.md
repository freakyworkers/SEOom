# AWS 스토리지 비용 가이드

**항목:** EBS 스토리지 (EC2 인스턴스 디스크)

---

## 💰 스토리지 비용 정보

### Free Tier 대상인 경우

**무료 제공량:**
- 월 30GB의 범용 SSD (gp2 또는 gp3) 스토리지
- 최대 20GB까지 무료로 사용 가능

**20GB 선택 시:**
- ✅ **비용:** 무료 (Free Tier 범위 내)
- ✅ **상태:** 문제 없음

### Free Tier 대상이 아닌 경우

**서울 리전 기준 (gp3):**
- **20GB 비용:** 약 월 2,000원~3,000원
- **GB당 비용:** 약 월 100원~150원

**비용 계산 예시:**
- 20GB: 약 월 2,000원~3,000원
- 30GB: 약 월 3,000원~4,500원
- 50GB: 약 월 5,000원~7,500원

---

## 📋 스토리지 크기 추천

### 최소 권장 크기

- **20GB**: 최소 권장 (프로젝트 파일, 데이터베이스 기본 용량)
- **30GB**: 여유 있음 (로그 파일, 업로드 파일 포함)
- **50GB**: 충분함 (대용량 파일, 이미지 등)

### 선택 가이드

| 용도 | 추천 크기 | Free Tier | 월 예상 비용 (Free Tier 아님) |
|------|----------|-----------|------------------------------|
| 테스트/개발 | 20GB | ✅ 무료 | 약 2,000원~3,000원 |
| 소규모 운영 | 30GB | ✅ 무료 | 약 3,000원~4,500원 |
| 중규모 운영 | 50GB | ❌ 유료 | 약 5,000원~7,500원 |

## 💡 비용 절약 팁

### 1. 필요한 만큼만 할당

- 처음에는 20GB로 시작
- 부족하면 나중에 늘릴 수 있어요
- 줄이는 건 안 되지만, 늘리는 건 가능해요

### 2. 사용하지 않는 파일 정리

- 로그 파일 정기적 삭제
- 불필요한 업로드 파일 삭제
- 데이터베이스 백업 파일 관리

### 3. 스토리지 모니터링

- AWS 콘솔에서 사용량 확인
- 알림 설정 (80% 사용 시 알림)

---

## ⚠️ 주의사항

### 스토리지 확장

- ✅ **늘리기**: 가능 (스냅샷 생성 후 볼륨 크기 변경)
- ❌ **줄이기**: 불가능 (볼륨 크기는 줄일 수 없어요)

### 스토리지 삭제

- 인스턴스를 삭제할 때 스토리지도 함께 삭제됨
- ⚠️ **중요**: 삭제하면 데이터가 사라져요!
- 백업이 필요하면 스냅샷을 먼저 생성하세요

## 📊 스토리지 사용량 확인

### AWS 콘솔에서 확인

1. EC2 콘솔 접속
2. **"볼륨"** 메뉴 클릭
3. 사용 중인 볼륨 확인
4. 사용량 확인

### 서버에서 확인

```bash
# 디스크 사용량 확인
df -h

# 특정 디렉토리 사용량 확인
du -sh /var/www/seoom
```

## 🆘 질문

**Q: 20GB가 부족하면 어떻게 하나요?**
A: 
- 볼륨 크기를 늘릴 수 있어요 (스냅샷 생성 후)
- 또는 추가 볼륨을 연결할 수 있어요

**Q: 스토리지 비용이 너무 비싸요.**
A: 
- 불필요한 파일 정리
- 로그 파일 자동 삭제 설정
- 사용하지 않는 볼륨 삭제

**Q: Free Tier가 끝나면 어떻게 되나요?**
A: 
- 스토리지 비용이 발생하기 시작해요
- 20GB면 약 월 2,000원~3,000원 정도
